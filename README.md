# 동구 · 부산광역시

> 동구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-dong.github.io/**

동구는 부산항 북항을 끼고 바다와 산비탈이 맞닿아 형성된, 부산 원도심의 관문 같은 곳입니다. 조선시대 초량왜관이 자리했고 개항 이후 부두와 철도가 들어서면서 사람과 물자가 가장 먼저 드나들던 무역·교통의 첫 관문 역할을 해 왔습니다. 한국전쟁 때 전국에서 몰려든 피난민이 산비탈에 판잣집을 지으며 만들어진 수정·초량의 산복도로 마을은 오늘날 이바구길과 168계단 같은 도보 관광 명소로 되살아났습니다. KTX와 SRT가 종착하는 부산역, 화교들이 모여 만든 상해거리 차이나타운, 그리고 북항 재개발로 솟아오르는 초고층 스카이라인이 한 동네 안에서 어우러지는 독특한 동네입니다. 좁고 가파른 골목마다 근현대 부산의 이주와 생활사가 켜켜이 쌓여 있어 '이야기'가 곧 자산이 된 지역입니다.
